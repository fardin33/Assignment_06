## Project Name : DigiTools Platform

## About Project :

DigiTools is a modern digital marketplace landing page designed to sell premium tools and templates. It is built using React and features an interactive product listing and dynamic cart system. The project uses a responsive UI, pricing tables, and a clean 3-step onboarding process to simplify the user experience.

## Technologies Used :

1. Html
2. Tailwind CSS , DaisyUI
3. JavaScript
4. React.js
5. React-Toastify (NPM Package)
6. JSON (For Product Data)

## 3 Features About DigiTools Project :

1. Dynamic Product Management : An interactive catalog that allows users to browse premium digital tools and seamlessly manage items through a real-time shopping cart system.

2. Modern Responsive UI : A sleek, professional landing page featuring high-impact sections like a hero area, 3-step onboarding, and detailed pricing tables, all optimized for various screen sizes.

3. Intuitive User Experience : A streamlined flow focused on conversion, utilizing clear call-to-action buttons, categorized product views, and a simplified checkout interface.
